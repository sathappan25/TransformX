How The Math Works

Translation shifts points in 3D: (x', y', z') = (x + tx, y + ty, z + tz).

3D rotation uses one matrix for each axis (X, Y, Z), then combines them.

Scaling changes object size per axis: (x', y', z') = (sx * x, sy * y, sz * z).

Perspective projection maps 3D to canvas using screen scale = cameraDistance / (cameraDistance + z).
